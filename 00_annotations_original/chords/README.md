## History

- Basis are the annotations by Cho and Bello: git://github.com/tmc323/Chord-Annotations.git
- 2026/04: (Stefan Balke and Arda Özgün)
    - converted the file format

## Format Specification

- semicolon-separated file
- temporal resolution: 10 ms
- Column 1 (t_start): start position in seconds
- Column 2 (t_end): end position in seconds
- Column 3 (chord): chord label
